# 🌤️ SkyCast – Weather Forecast Application

SkyCast is a modern, responsive weather forecast web application built with HTML, CSS, JavaScript, and Bootstrap 5. It fetches real-time weather data and displays daily and weekly forecasts, along with current temperature, humidity, and wind speed for any city worldwide.

## 📌 Features
- Current Weather : Displays temperature, humidity, and wind speed for a selected city.
- Weekly Forecast : Shows a 5-day forecast with average temperature, humidity, and wind speed.
- Search with Autocomplete : City search with suggestions powered by OpenWeatherMap’s geocoding API.
- Dynamic Footer Cities : Add multiple cities to track their weather in the footer section.
- Responsive Design : Works seamlessly across desktops, tablets, and mobile devices.
- Interactive UI : Collapsible forecast table for detailed info
- Modal for adding more cities : "See More" button to expand footer city cards

## 🛠️ Technologies Used
- Frontend: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
- Icons: Bootstrap Icons
- Weather API: OpenWeatherMap API
- Geocoding API: OpenWeatherMap Geocoding

## 💡 Usage
- Search for a city using the search input bar.
- Click a suggested city to update weather information.
- View daily forecasts in the collapsible info section.
- Add cities to the footer using the "Add More" modal.
- Toggle footer display using the "See More" / "See Less" button.
